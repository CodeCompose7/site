## 강의 개요

이 강의는 머신러닝 모델을 실제 서비스에 안정적으로 배포하고 운영하고자 하는 분들을 위한 MLOps 실전 과정입니다.

### 학습 목표

- MLOps 개념과 머신러닝 라이프사이클 이해
- 데이터 파이프라인과 학습 파이프라인 설계
- 모델 저장소 및 버전 관리 도입
- 모델 배포와 모니터링 전략 수립

### 강의 구성

**1부: MLOps와 파이프라인 기초**
- MLOps 등장 배경과 필요성
- 데이터 파이프라인 vs 학습 파이프라인
- 주기적인 재학습(Continuous Training) 개념

**2부: 도구와 플랫폼**
- MLflow 또는 Weights & Biases를 활용한 실험 관리
- 모델 레지스트리와 버전 관리
- Kubeflow / Airflow 등을 활용한 워크플로 오케스트레이션 개요

**3부: 모델 배포와 모니터링**
- REST API 기반 모델 서빙 (예: FastAPI + Docker)
- 모델 성능 드리프트 모니터링
- 롤백 전략과 A/B 테스트 개요

### 강의 방식

- **온라인/오프라인**: Zoom 또는 대면 강의
- **실습 중심**: 예제 데이터와 모델을 활용한 파이프라인 구축 실습
- **1:1 피드백**: 수강생 프로젝트 MLOps 구조 리뷰

### 수강 대상

- 머신러닝 모델을 이미 만들어봤거나, 서비스에 적용해 보고 싶은 분
- 데이터 사이언티스트, ML 엔지니어, 백엔드 개발자

### 선수 지식

- Python과 머신러닝 기본 개념
- Git 및 간단한 DevOps 개념(권장)
